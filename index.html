Work

Some stuff
I've been working on

Mindset

Mindset is a single-page, full-stack clone of Headspace - an app that provides an approachable and visually engaging introduction to meditation. It is built using React, Redux, Ruby on Rails, and Javascript.

CompYOUter

CompYOUter is a project that explores the field of Human Computer Interaction. It uses machine learning libraries Tensorflow and Posenet to track a user's movements and create an interative experience.

Disarray

DisArray is a fast-paced, multi-player game inspired by Boggle! It is built with the MERN stack (MongoDB | Express | React | Node.js) and hosted on Heroku.
